# HealthChain AI - Blockchain + IA para Salud Digital

Este repositorio contiene la implementación del proyecto Expediente X: Identidad Digital en Blockchain + IA Predictiva.

## Estructura del Proyecto

- `blockchain/`: Código relacionado con Hyperledger Fabric, smart contracts y DID.
- `ai/`: Motor de IA con TensorFlow/PyTorch para análisis predictivo.
- `frontend/`: Aplicación móvil y web con React Native y Next.js.
- `backend/`: API Gateway, servicios backend y integración con IPFS/PostgreSQL.
- `docs/`: Documentación adicional, incluyendo el Lean Canvas y evaluaciones.
- `tests/`: Pruebas unitarias e integración.

## Próximos Pasos

1. Configurar el entorno de desarrollo para Hyperledger Fabric.
2. Implementar prototipo básico de DID.
3. Desarrollar modelo de IA simple con datos sintéticos.
4. Crear prototipo de app móvil básica.
